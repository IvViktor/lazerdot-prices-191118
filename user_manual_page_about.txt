І. Блок слайдера
На странице сайта О КЛИНИКЕ блок слайдера выглядит как набор страниц с текстовым полем в левой части страницы и фотграфией в правой части. Страницы слайдера автоматически прокручиваються с заданым интервалом. Кроме того существует возможность выбора нужной интересующей странички слайдера вручную.
Блок слайдера напоняеться страничками из панели вордпресса через WYSIWYG редактор размещенный на вкладке "Блок слайдера". Для создания позиций слайдера применяються специальние "шорткаты" - ключевые слова в квадратных скобках. Содержимое шортката помещаеться между открывающим и закрывающим тегами (напр. [slideritem] ??СОДЕРЖИМОЕ?? [/slideritem]
Порядок создания позиций слайдера следующий:
1. Создаем позицию слайдера при помощи шорткатов [slideritem] и [/slideritem]. Содержимое позиции будем помезать между ними.
2. Внутри шортката [slideritem] для создания текстового блока вставляем шорткат [slideritemtext header="??ЗАГОЛОВОК??"]  [/slideritemtext]. В параметр header вставляем заголовок текстового блока. После этого внутри шортката  [slideritemtext] вставляем текст который будет отображаться на позиции слайдера.
3. Внутри шортката [slideritem] для создания блока с фотографией вставляем шорткат [slideritemphoto]  [/slideritemphoto]. Внутрь шортката [slideritemphoto] вставляем фотографию средствами редактора вордпресс.
4. Повторяем пункты 1-3 для каждой создаваемой позиции слайдера.


ІІ. Блок "ПРЕИМУЩЕСТВА КЛИНИКИ"
Блок "ПРЕИМУЩЕСТВА КЛИНИКИ" выглядит как набор текстовых блоков размещенных построчно, по 2 блока на строку, внутри декоративной рамки. Текстовые блоки наполняються из панели вордпресса через WYSIWYG редактор размещенный на вкладке "Блок "Преимущеста клиники". Для создания текстовых блоков применяються шорткаты блоков [advantageitem] помещенные внутри строк [advantagerow].
Порядок наполнения блока следующий:
1. Создаем сроку при помои шорткатов [advantagerow] и [/advantagerow].
2. Внутри шортката [advantagerow] для созданий текстового блока вставляем шорткат [advantageitem header=""] [/advantageitem]. В поле header вставляем текст заголовка блока. Внутрь шортката [advantageitem] вставляем текст блока.
3. Аналогичную операцию проводим для создания второго блока в строке. ВНИМАНИЕ: строка моедт содержать не более двух текстовых блоков за раз.
4. Повторяем операции 1-3 для создания других строк и текстовых блоков.

ІІІ. Блок "Ведущие врачи"
Блок "Ведущие врачи" выглядит как набор раскрывающихся позиций с информацией о врачах. Позиции с информацией о врачах создаються из панели вордпресса через WYSIWYG редактор размещенный на вкладке "Блок "Ведущие врачи". Для создания позиций о врачах применяються шорткаты позиций [doctoritem], шорткаты краткой информации о докторе [doctorintro], шорткаты подробной информации о докторе для создания раскрывающегося списка.
Порядок создания позиций о врачах следующий:
1. Создаем тело позиции о враче с помощью шорткатов [doctoritem] и [/doctoritem].
2. Внутрь шортката [doctoritem], для создания кнопки с фото и именем врача вставляем шорткат [doctorintro name="" spec=""] [/doctorintro]. В поле name вводим имя доктора, а в поле spec вводим его специальность. Внуть шортката [doctorintro] вставляем фото доктора средствами редактора вордпресс. (Фото доктора уже должно быть отредактировано и иметь окончательный вид).
3. Внутрь шортката [doctoritem], для создания раскрывающегося блока с информацией о враче вставляем шорткат [doctordescr] [/doctordescr].
4. Внутрь шортката [doctordescr], вставляем шорткат [doctoreducation] [/doctoreducation] для создания блока с данными об образовании. Внутрь шортката [doctoreducation] текст вставляем в виде ненумерованого списка средствами редактора вордпресс.
5. Внутрь шортката [doctordescr], вставляем шорткат [doctoradditional] [/doctoradditional].
6. Внутрь шортката [doctoradditional], вставляем шорткаты [doctorexperience] [/doctorexperience], а также [doctorcertificates] [/doctorcertificates] для блоков Опыт работи и Сертификаты соответственно. Данные внутри этих шорткатов выполняем в виде ненумерованого списка средствами редактора вордпресс.
